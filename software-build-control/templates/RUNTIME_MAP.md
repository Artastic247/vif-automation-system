# RUNTIME_MAP

## Purpose
Map UI/workflow to real runtime objects and backend proof.

## Required fields
- Runtime object
- State model
- Allowed actions
- Protected actions
- Database table/RPC/function
- RLS/security rule
- Backend read proof
- Backend write proof
- Audit/evidence record

## Gate controlled
Runtime/backend gate.

## Gate decision
PASS / HOLD / BLOCKED
