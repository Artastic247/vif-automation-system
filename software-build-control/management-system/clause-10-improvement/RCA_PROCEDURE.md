# RCA_PROCEDURE

## Purpose
Define structured root cause analysis for software-build and management-system failures.

## Scope
Systemic NCs, repeated failures, false PASS, prompt/tool failure, validator failure, release/rollback failure, security/RLS risk and maturity overstatement.

## Owner/agent
RCA/CAPA Specialist with affected process owner and relevant specialist.

## Inputs
Linked NC, evidence, containment status, correction record, audit/validation findings and affected artefacts.

## Activities/method
Define problem statement, source of issue, evidence, containment taken, direct cause, systemic cause, escaped-detection cause, maturity-level cause, interface/control-point failure, prompt/tool contribution, agent/skill contribution, process/procedure gap, validation/checker gap, risk-control failure, root-cause method used, corrective action required, verification of correction and effectiveness-review requirement.

## Outputs/records
RCA record, cause classification, corrective-action requirement and effectiveness-review trigger.

## Evidence required
Evidence reviewed, cause logic, method used, rejected assumptions, linked artefacts and action decision.

## Linked process
Clause 10 RCA.

## Linked gate
Root cause gate.

## MLA / maturity dependency
Systemic RCA findings may trigger maturity downgrade or block maturity upgrade.

## Clause 9 input
Audit finding, MLA failure, validator false negative, evidence audit issue or release audit issue.

## Clause 10 output
Root cause, CAPA requirement, validator/procedure/prompt/skill update need.

## PASS/HOLD/BLOCKED rules
PASS when cause is evidence-supported. HOLD when analysis incomplete. BLOCKED when root cause is guessed or containment absent for high-risk issue.

## Escalation
Escalate unresolved systemic cause, repeated failure or unsafe customer/security impact.

## Management-review input
Systemic RCA trends feed management review.

## Continual-improvement input
RCA outputs drive process, prompt, skill, validator and knowledge updates.

## Update trigger
RCA required decision, repeat failure, ineffective CAPA or audit finding.
