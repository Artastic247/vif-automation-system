# OBJECTIVES_REGISTER

## Purpose
Set measurable objectives for control-system effectiveness.

## When to use
Planning and management review.

## Owner/tool
System owner.

## Inputs
Risks, performance data, defects, recovery goals.

## Activities or fields
| Objective | Measure | Target | Owner | Current result | Action | Status |
|---|---|---|---|---|---|---|

## Outputs/records
Objectives and performance evidence.

## Gate controlled
Objectives/performance gate.

## PASS/HOLD/BLOCKED rules
PASS if objectives are measurable and reviewed. HOLD if target/owner missing. BLOCKED if objective affects release safety and is uncontrolled.

## Update trigger
Management review, audit, or performance trend.
