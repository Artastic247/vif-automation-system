# SKILL_TEMPLATE

## Purpose
Define a reusable skill in controlled format.

## When to use
When creating or revising a skill.

## Owner/tool
Skill owner.

## Inputs
Skill ID; trigger; purpose; when to use; when not to use; required inputs; gate controlled.

## Activities or fields
Skill ID; trigger; purpose; when to use; when not to use; required inputs; method steps; output format; quality checks; failure modes; gate controlled; revision history.

## Outputs/records
Released or draft skill record.

## Gate controlled
Skill release gate.

## PASS/HOLD/BLOCKED rules
PASS if method, checks, outputs, and failure modes are clear. HOLD if untested. BLOCKED if it allows uncontrolled work.

## Update trigger
Skill creation, pilot, release, revision, deprecation.
