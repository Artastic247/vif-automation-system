# GATE_REVIEW_REGISTER

## Purpose
Record gate reviews and decisions.

## When to use
At intake, baseline, job-card, verification, rollout, release, and improvement gates.

## Owner/tool
Gate reviewer.

## Inputs
Gate criteria, evidence, risks, open actions.

## Activities or fields
| Gate | Job card | Evidence | Decision | Conditions | Reviewer | Date |
|---|---|---|---|---|---|---|

## Outputs/records
Gate decision record.

## Gate controlled
All gates.

## PASS/HOLD/BLOCKED rules
PASS if evidence satisfies criteria. HOLD if incomplete. BLOCKED if unsafe.

## Update trigger
Every formal gate decision.
