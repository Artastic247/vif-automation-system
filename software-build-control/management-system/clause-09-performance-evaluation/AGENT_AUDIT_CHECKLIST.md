# AGENT_AUDIT_CHECKLIST

## Purpose
Audit agent/specialist competence, authority and scope adherence.

## Scope
All assigned agents and specialist roles.

## Owner/agent
Agent Audit Specialist.

## Inputs
Assignment matrix, outputs, escalation records, maturity and audit history.

## Activities/method
Check role definition, competence evidence, prohibited work boundaries, escalation usage, evidence quality and scope compliance.

## Outputs/records
Agent audit record and findings.

## Audit criteria
Agents must operate only within approved scope and maturity.

## Evidence required
Assignment record, work evidence, escalation evidence, approvals.

## MLA / maturity dependency
Higher maturity roles require audit evidence and effectiveness review.

## Linked process
Clause 9 agent audit.

## Linked gate
Agent release gate.

## PASS/HOLD/BLOCKED rules
PASS if competence and controls are evidenced. HOLD if evidence incomplete. BLOCKED if agent exceeded authority or caused unsafe action.

## Escalation
Unauthorized automation, release or customer-facing claim.

## Management-review input
Competence and authority risks feed management review.

## Update trigger
Role change, finding or repeated issue.
