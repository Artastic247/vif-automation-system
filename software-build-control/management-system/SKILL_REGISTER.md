# SKILL_REGISTER

## Purpose
Register reusable skills and their lifecycle.

## When to use
Before using repeatable process, audit, verification, release, or problem-solving skills.

## Owner/tool
System owner and skill owner.

## Inputs
Skill definition, trigger, controlled gates, evidence requirements.

## Activities or fields
| Skill ID | Trigger | Purpose | When to use | When not to use | Required inputs | Output format | Gate controlled | Status | Revision history |
|---|---|---|---|---|---|---|---|---|---|

## Outputs/records
Controlled skill register.

## Gate controlled
Skill governance gate.

## PASS/HOLD/BLOCKED rules
PASS if skill is approved and current. HOLD if under review. BLOCKED if obsolete or unsafe.

## Update trigger
New skill, failed use, revision, or lesson learned.
